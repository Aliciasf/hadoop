Hadoop Issue Tracking

Hadoop tracks both bugs and enhancement requests here using Apache JIRA. We welcome input, however, before filing a request, please make sure you do the following:

  • Search the JIRA database.
  • Check the user mailing list, both by searching the archives and by asking questions.
